# Task Manager API 🚀

## Descripción

Esta es una Web API robusta desarrollada con **Node.js** y **Express**. Sirve como el núcleo del sistema de gestión de tareas, permitiendo realizar operaciones CRUD básicas (Crear, Leer y Eliminar) a través de endpoints RESTful. Está diseñada para ser desplegada en plataformas como **Render** y maneja configuraciones de entorno dinámicas.

## Estructura de Archivos (Tree)

```text
task-manager-api/
├── node_modules/       # Dependencias del proyecto (no se sube a GitHub)
├── server.js           # Archivo principal del servidor y endpoints
├── package.json        # Configuración del proyecto y scripts
├── package-lock.json   # Registro de versiones de dependencias
└── .gitignore          # Archivo para excluir node_modules de Git
```
